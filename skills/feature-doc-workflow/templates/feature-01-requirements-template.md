## 사용자 시나리오

### 시나리오 1. 기본 성공 흐름

1. 사용자가 진입
2. 핵심 액션 수행
3. 기대 결과 확인

### 시나리오 2. 대체 진입 흐름

1. 다른 경로로 진입
2. 동일 목적 수행
3. 결과 확인

### 시나리오 3. 예외 또는 제한 상황

1. 사용자가 제한 상황 진입
2. 시스템이 안내
3. 대체 흐름 진행
