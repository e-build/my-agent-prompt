## 오픈 포인트

1. 추가 합의가 필요한 항목
2. API 정의 전에 닫아야 할 항목
3. 후속 기능과 연결되는 항목

## API 정의 전 체크 항목

- 클라이언트와 서버가 분기해야 할 상태가 충분히 정리되었는지
- 예외 상황이 사용자 행동 수준으로 정리되었는지
- 권한 / 노출 / 만료 기준이 모호하지 않은지
- 화면정의서와 충돌하는 정책이 없는지
