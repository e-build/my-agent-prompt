# Shopl Jira Briefing Examples

## 기본 호출

- `내 Jira 이슈 트리로 정리해줘`
- `jimmy 한테 열려있는 이슈 리스트업해줘`
- `close 상태가 아닌 이슈를 상위 이슈 기준으로 묶어줘`
- `QA 상태 이슈만 부모-자식 형태로 보여줘`

## 실질 미종결

- `완료/해결됨/Ready PROD 빼고 아직 액션할 이슈만 보여줘`
- `실질 미종결 이슈만 트리로 정리해줘`

## 기대 출력

```text
조회 기준: assignee = currentUser(), 실질 미종결(statusCategory != Done)

- SH-15761
  - SH-18440 [진행 중][스토리] - [Backend] 근태 종합 리포트 고도화
    - SH-20624 [DEV][하위 작업] - [9A] 범위 분석
    - SH-20625 [DEV][하위 작업] - [9D] 계산 모델 적용

- 상위 이슈 없음
  - OP-1257 [해야 할 일][버그] - 자동퇴근 날짜 인식 이슈
```

## 후속 요청

- `상태별 개수도 추가해줘`
- `SH-18440 하위만 따로 정리해줘`
- `부모 제목까지 붙여줘`
- `Close만 제외한 전체 목록으로 다시 보여줘`
