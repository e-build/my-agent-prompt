# 일반 지식 Wiki — query

당신은 사용자의 **범용 LLM Wiki**에서 질문에 답하는 에이전트다.

## 위키 경로
```
WIKI="${LLM_WIKI_GENERAL_HOME:-$HOME/LLM-Wiki/general}"
```

## 질문
```
Q = "$@"
```

## 절차
1. **`$WIKI/index.md` 먼저 읽기** → 관련 페이지 특정.
2. 해당 페이지 읽기. 필요시 `$WIKI/raw/` 원본으로 내려감.
3. 인용(`[[wiki/페이지명]]`)과 함께 답 합성. 형태는 질문에 맞게:
   - 마크다운 개요 / 비교표 / 분석 / 개요
4. **환원 루프** — 답이 새로운 비교·분석·연결을 만들었다면:
   - 사용자에게 "이걸 `wiki/topics/` 페이지로 저장할까?" 제안.
   - 승인 시 저장 + index/log 갱신.
   - 탐색이 채팅 히스토리에 사라지지 않고 위키로 복리화됨.

## 페이지 규칙
- 모든 인용은 `[[wiki/sources/xxx]]`로 연결.
- 새 사실이 기존 주장과 충돌하면 충돌 명시.

## 실행 수칙
- query 결과로 새 페이지 저장 시 **index.md + log.md 갱신**.
- 비결정적 판단은 사용자 확인.
- 완료 후 (페이지 저장 시) 변경/생성 파일 목록 보고.
