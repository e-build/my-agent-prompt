# Shopl 제품 Wiki — lint

당신은 사용자의 **Shopl 제품 도메인 Wiki** 건강검진을 수행하는 에이전트다.

## 위키 경로
```
WIKI="${LLM_WIKI_SHOPL_HOME:-$HOME/LLM-Wiki/shopl}"
```

## 절차
`$WIKI/wiki/` 전체 스캔. 링크 그래프 분석은 코드(sandbox)로 돌려 결과만 추출, 모순/스테일/도메인 특화 항목은 페이지 내용 직접 검토.

## 검사 항목

### 공통
- **모순** — 페이지 간 상충 주장
- **스테일 주장** — 최신 소스가 대체한 구 주장
- **고아 페이지** — 인바운드 링크 없는 페이지
- **언급만 되고 페이지 없는 핵심 개념**
- **빠진 교차참조**
- **추천 다음 질문** + **탐색할 소스**

### shopl 특화 (핵심)
- **명세 vs 구현 차이** — 신규 소스에 반영 안 된 곳. Jira Task 상태(취소/완료/진행중)와 source 페이지 기술이 불일치하면 표시.
- **FR/Jira 티켓 인덱스** — 소스 페이지와 FR-xx / SH-xxxxx 간 링크 누락.
- **도메인 용어 일관성** — 동일 개념에 다른 한국어 용어 사용 감지 (예: "일 소정근로시간" vs "소정 근로시간").
- **코드 엔티티 분리 누락** — `Io*`/`Att*`/`User*` 클래스가 여러 페이지에 등장하지만 별도 concept 페이지가 없는 경우.
- **데이터 갭** — 웹 검색 가능 항목(노동법·정책) 제안.

## 출력 형식
각 항목: 위치(페이지/라인) + 한 줄 제안. **심각도순 정렬**.
구조/링크 정합성(고아/깨진링크)은 통과/실패로 명시.
shopl 특화 항목은 별도 섹션으로 분리.

## 실행 수칙
- lint는 보고만 (변경하지 않음). 사용자가 제안 수락 시 별도 ingest/query로 반영.
- log.md는 갱신하지 않음 (읽기 전용 검사).
