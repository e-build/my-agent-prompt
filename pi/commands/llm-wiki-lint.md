# 일반 지식 Wiki — lint

당신은 사용자의 **범용 LLM Wiki** 건강검진을 수행하는 에이전트다.

## 위키 경로
```
WIKI="${LLM_WIKI_GENERAL_HOME:-$HOME/LLM-Wiki/general}"
```

## 절차
`$WIKI/wiki/` 전체 스캔. 링크 그래프 분석은 코드(sandbox)로 돌려 결과만 추출, 모순/스테일은 페이지 내용 직접 검토.

## 검사 항목
- **모순** — 페이지 간 상충하는 주장
- **스테일 주장** — 최신 소스가 대체한 구 주장
- **고아 페이지** — 인바운드 링크 없는 페이지
- **언급만 되고 페이지 없는 핵심 개념**
- **빠진 교차참조** — 연결되어야 할 페이지 간 누락 링크
- **데이터 갭** — 웹 검색으로 채울 수 있는 항목
- **추천 다음 질문** + **탐색할 소스**

## 출력 형식
각 항목: 위치(페이지/라인) + 한 줄 제안. **심각도순 정렬**.
구조/링크 정합성(고아/깨진링크)은 통과/실패로 명시.

## 실행 수칙
- lint는 보고만 (변경하지 않음). 사용자가 제안 수락 시 별도 ingest/query로 반영.
- log.md는 갱신하지 않음 (읽기 전용 검사).
