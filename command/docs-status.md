# 문서화 현황 대시보드

docs/ 전체 현황을 분석하여 대시보드 형태로 출력.

---

## 상태 판단 기준

### Feature 상태
| 상태 | 기준 |
|------|------|
| ✅ 완료 | index.md에 TODO 없음, 문서 충분 |
| 🚧 진행중 | index.md에 TODO 있음 |
| ⚠️ 부실 | 문서 1-2개뿐, 내용 부족 |
| 🔴 미시작 | index.md만 있거나 빈 디렉토리 |

### 문서 건강도
| 상태 | 기준 |
|------|------|
| 🟢 건강 | 최근 수정, 내용 충실 |
| 🟡 주의 | 6개월+ 미수정 |
| 🔴 위험 | 1년+ 미수정, 내용 부족 |
