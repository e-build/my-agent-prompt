## 문서 목적

- 프로젝트 디자인의 근본 규칙과 이유 정의
- 모든 디자인 결정의 기준 역할
- 팀 내 디자인 언어 공유 목적

## 핵심 원칙

> 프로젝트 디자인의 핵심 가치 3~5가지를 문장으로 정의합니다.

예시:
1. **기능 우선**: 장식보다 사용성. 모든 디자인 요소는 목적이 있어야 한다
2. **일관성**: 같은 것은 같게, 다른 것은 다르게
3. **접근성**: 모든 사용자가 동등하게 사용할 수 있어야 한다
4. **미니멀리즘**: 필요 없는 것은 없다
