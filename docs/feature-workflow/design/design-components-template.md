## 컴포넌트별 정의

### Button

**역할**: 사용자 액션을 트리거하는 컨트롤

**변형**:

| 변형 | 용도 |
|------|------|
| Primary | 주요 CTA |
| Secondary | 보조 액션 |
| Ghost | 가벼운 액션 |
| Danger | 위험한 액션 |

**상태**:

| 상태 | 설명 |
|------|------|
| 기본 | Normal |
| Hover | 마우스 오버 |
| Active | 클릭 중 |
| Disabled | 비활성 |
| Loading | 처리 중 |

**크기**:

| 크기 | 높이 | 패딩 | 폰트 |
|------|------|------|------|
| sm | — | — | — |
| md | — | — | — |
| lg | — | — | — |

---

### TextInput

**역할**: 사용자 텍스트 입력

**상태**:

| 상태 | 설명 |
|------|------|
| 기본 | 빈 입력 가능 상태 |
| Focused | 포커스됨 |
| Filled | 값 입력됨 |
| Error | 유효성 오류 |
| Disabled | 비활성 |
